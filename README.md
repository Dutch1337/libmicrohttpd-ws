# LIBMICROHTTPD WITH WEBSOCKET
This is a first beta release of libmicrohttpd with websocket support.

# GET START

To install the library :

    mkdir build
    cd build
    cmake ..
    make microhttpd_install
    make run_main

Go to localhost:9090 [localhost:9090](http://localhost:9090)

# DEPENDENCIES
 - autoconf
 - automake
 - libgcrypt11-dev
 - libtool
 - libcurl
